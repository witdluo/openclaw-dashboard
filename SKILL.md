# OpenClaw Dashboard - 专业可视化仪表盘

一个现代化的 OpenClaw 管理面板，实时展示 Agent 状态、定时任务、资源监控等。

---

## 🎨 界面特点

- **暗色主题** - 深色渐变背景，护眼舒适
- **玻璃态设计** - 毛玻璃效果，现代感十足
- **渐变色彩** - 丰富的渐变色，视觉层次分明
- **流畅动画** - 悬停、加载动画，交互反馈
- **响应式布局** - 支持桌面和移动端

---

## 🚀 快速启动

```bash
cd ~/.openclaw/workspace/openclaw-dashboard
npm run dev
```

访问：http://localhost:3000

---

## 📊 功能模块

| 模块 | 功能 |
|------|------|
| **Agents** | 6 个 Agent 实时状态、模型配置 |
| **定时任务** | 18 个任务调度、执行历史 |
| **资源监控** | CPU/内存/磁盘趋势图表 |
| **消息记录** | 各渠道消息收发记录 |
| **错误日志** | 错误/警告分级展示 |

---

## 🛠️ 技术栈

- React 18 + TypeScript
- Vite (构建)
- Tailwind CSS (样式)
- Recharts (图表)
- TanStack Query (数据)
